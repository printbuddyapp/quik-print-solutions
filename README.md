A simple Windows desktop app to print **2 ID photos on one A4 page**.

## Features
- Select exactly 2 photos
- Auto rotate portrait images
- Fit / Fill modes
- Swap images
- Print using selected printer
- CTRL + P shortcut

## Tech
- Python
- Tkinter
- Pillow
- ReportLab
- Windows Print API

## Use Case
Ideal for:
- Stationery shops
- Cyber cafes
- Home / Office ID printing
